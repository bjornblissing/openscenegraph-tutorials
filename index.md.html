**OpenSceneGraph Tutorials** Preface ======= These tutorials were written to help OpenSceneGraph beginners get a quick start. It is also meant to help intermediate users with common questions. Introduction ============ Source code ----------- * Downloading the OSG source code * Downloading the OSG example data * Downloading the tutorial source code * Third party libraries Building from source -------------------- * Windows * Linux * Mac * Android * IOS * Web Basic ===== * [Basic Geometry](basic_geometry/basic_geometry.md.html) * Textured objects * Reading a 3D-model file * Scene graph basics * Dynamic scene objects * Rendering text * Simple shaders Intermediate ========== * Interpreting the statistics view * Reference pointers * Slave cameras * Building your own camera controler * Picking objects * Animation * Adding and removing objects dynamically Advanced ======== * Organizing your scenegraph for maximum performance * Occluders * Impostors * Level of detail * Database paging Other resources =============== There are three public mailing lists; *osg-users* for general support and development discussion, *osg-news* for announcements of releases and *osg-submissions* for submitting bug fixes or other code.
